Temat:
Kształtowanie się wartości poszczególnych kryptowalut w czasie, z uwzględnieniem
wydarzeń jakie miały miejsce, z podziałem na kategorie wydarzeń (np. gospodarka,
popkultura itp.). Analiza siły oddziaływania poszczególnych wydarzeń i kategorii
wydarzeń oraz różnic w trendach pomiędzy kryptowalutami. 

System: Windows 11
.NET 6.0
Potrzebny jest docker compose oraz zmiana portów dla xampp na 8012 żeby uniknąć konfliktów
XAMPP Control Panel v 3.3.0
PHP 8.2.0

Aplikacja pokazuje poszczególne dane o cenach kryptowalut oraz ich zmiana za pomocą grafików. Również jest uwierzetylnianie co pozwala na zobaczenie tabeli przyszłościowych cen na Ethereum. Jest komunikacja z REST, SOAP oraz jest używany Docker. Automatyczny export do xml oraz yaml przy wyświetlaniu tabeli lub import do bazy danych. Zczytanie z JSON oraz XML.
